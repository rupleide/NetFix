# NetFix v1.0.6 - Changelog

## 🎉 Новые возможности

### Онбординг и установка
- **Добавлен выбор способа установки**: Теперь при первом запуске можно выбрать автоматическую установку (всё настроится за 15 секунд) или ручную установку с пошаговыми инструкциями
- **Улучшен экран автоматической установки**: Добавлен красивый прогресс-бар, лог установки в реальном времени и процент выполнения
- **Восстановлен полный флоу онбординга**: Приветствие → Описание → Дисклеймер → Выбор способа установки → Установка

### Telegram-канал проекта
- **Добавлена секция Telegram-канала в настройках**: Красивая зелёная карточка с информацией о канале и кнопкой для подписки
- **Добавлена ссылка на Telegram-канал в раздел "ССЫЛКИ"**: Быстрый доступ к каналу @NetFixRuBi прямо из настроек
- **Добавлена карточка "TgWsProxy на Android" в FAQ**: Специальная выделенная карточка с информацией о решении для Android и ссылкой на инструкцию в Telegram-канале
- **Deep link для Telegram**: Ссылки на канал открываются напрямую в приложении Telegram, минуя браузер (обход блокировок)

### Ссылки и навигация
- **Добавлена ссылка на GitHub репозиторий NetFix**: Теперь можно быстро перейти на страницу проекта
- **Убрана ссылка на GoodbyeDPI**: Заменена на более актуальную ссылку на NetFix GitHub

## 🎨 Улучшения интерфейса

### Дизайн и стили
- **Улучшен дизайн FAQ**: Более современные карточки с лучшими отступами и границами
- **Упрощена акцентная линия в ответах FAQ**: Теперь выглядит чище и профессиональнее
- **Изменён фон карточек ответов FAQ**: Более тёмный цвет (#1c1c1c) для лучшего контраста
- **Создан отдельный стиль GreenAccentBtn**: Зелёная кнопка с правильным hover-эффектом (темнее зелёный при наведении)
- **Улучшена карточка "Ручное решение"**: Стильная тёмная карточка с синей боковой полосой

### Кнопки и элементы управления
- **Исправлен хитбокс кнопки настроек**: Добавлен прозрачный фон, теперь вся область кнопки кликабельна, включая центр с иконкой
- **Создан отдельный стиль FlatBtnCentered**: Для кнопки закрытия настроек с центрированным контентом, не ломает другие кнопки

### Текст и типографика
- **Заменены все длинные тире (—) на запятые**: Улучшена читаемость текста во всём приложении
- **Удалены лишние пробелы перед запятыми**: Правильная типографика
- **Исправлен текст главной кнопки**: "Одна кнопка - скачает, настроит и запустит всё необходимое"

## 🐛 Исправления

### Установщик
- **Исправлена проблема с установкой .NET Runtime**: Установщик теперь корректно работает без файла dotnet-runtime (закомментирована строка, которая вызывала ошибку компиляции)

### Навигация и ссылки
- **Исправлен цвет ссылки на Telegram-канал**: Теперь синий, как и остальные ссылки в разделе "ССЫЛКИ"
- **Telegram deep links**: Ссылки открываются напрямую в приложении Telegram через протокол tg://

## 📦 Технические изменения

- **Обновлена версия приложения**: 1.0.5 → 1.0.6
- **Обновлена версия в установщике**: NetFix_Setup_v1.0.6
- **Множество коммитов для улучшения стабильности и качества кода**

## 📥 Установка

1. Скачайте `NetFix_Setup_v1.0.6.exe`
2. Запустите установщик от имени администратора
3. Следуйте инструкциям на экране
4. При первом запуске выберите автоматическую или ручную установку
