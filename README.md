## 📖 О проекте

В современных реалиях доступ к привычным сервисам вроде **Telegram**, **Discord** и другим часто ограничивается. Настройка обходных путей через `zapret`, `tg-ws-proxy` или конфигурацию прокси — задача не для каждого.

**NetFix** — это интуитивно понятная оболочка (GUI), которая автоматизирует весь процесс. Вам не нужно открывать консоль или редактировать конфиги. Скачал, нажал кнопку — и всё работает.

---

## ✨ Возможности

- ⚡ **Автоматизация "под ключ"** — скачивает и настраивает компоненты (zapret, proxy) за вас.
- 🔧 **Управление в один клик** — мгновенный запуск и остановка всех служб.
- 📡 **Диагностика системы** — автоматический поиск проблем и проверка доступности серверов.
- 🚀 **Оптимизация** — минимальное потребление ресурсов, работа в системном трее.
- 🔄 **Умные обновления** — проверка новой версии при запуске и установка в один клик.
- 🎨 **Современный интерфейс** — чистый дизайн на WPF с поддержкой тем.

## 🚀 Установка и использование

1. Перейдите в раздел **[Releases](https://github.com/rupleide/NetFix/releases/latest)**.
2. Скачайте актуальный файл `NetFix_Setup.exe`.
3. Запустите установщик (могут потребоваться права администратора).
4. В приложении нажмите кнопку **"Починить интернет"**.

> **Важно:** Для корректной работы рекомендуется отключить сторонние VPN-сервисы на время диагностики.
